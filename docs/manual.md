# 🔶Projeto: Enigma🔶 #
___
## Introdução ##
O trabalho que fomos requisitado para fazermos, foi programar uma maquína que criptografa ou descriptografar uma mensagem que enviaremos para o software, inspirado no filme "Jogo da Imitação".
___
## Como funciona? 😯 ##
* ## 🤫 Senha: ##
- A senha para poder realizar os procedimentos é 2871.
* ## Criptografar: ##
- Você deve inserir uma mensagem e o software irá fazer o código usando a cifra de césar.
* ## Descriptografar: ##
- Você deve inserir o código e o software irá traduzir a mensagem.
